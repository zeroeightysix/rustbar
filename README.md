## Compile & run

Assuming you have `git`, `cargo` and [gtk-layer-shell](https://github.com/wmww/gtk-layer-shell) installed:
```
git clone https://github.com/zeroeightysix/rustbar
cargo build --release
target/build/rustbar
```
(`cargo run` in development)
